# Portfolio Web - Edgar Iván Vázquez Segovia

Este repositorio contiene el **Frontend** de mi portafolio profesional. Es una aplicación moderna de una sola página (SPA) construida para mostrar mis proyectos y habilidades de manera dinámica, consumiendo datos directamente desde una API personalizada.

## 👤 Sobre el Autor
**Edgar Iván Vázquez Segovia** *Software Engineer | Full Stack Developer*

---

## 🚀 Tecnologías Utilizadas
* **Vue.js 3** (Composition API) - Framework principal.
* **Vite** - Herramienta de construcción y servidor de desarrollo ultra rápido.
* **Axios** - Cliente HTTP para la comunicación con el Backend.
* **JavaScript (ES6+)** - Lógica de la aplicación.
* **HTML5 & CSS3** - Estructura y diseño responsivo.

## 🔌 Arquitectura (Full Stack)
Este proyecto es la cara visual de un ecosistema desacoplado. Los datos que visualizas aquí son gestionados y servidos por mi API desarrollada en .NET 8.

* **Backend Repository:** [PortfolioAPI](https://github.com/elpublica/PortfolioAPI)

## 🛠️ Instalación y Ejecución Local

Para poner en marcha el proyecto localmente, ejecuta los siguientes comandos en tu terminal:

```bash
# 1. Clonar el repositorio
git clone [https://github.com/elpublica/PortfolioWeb.git](https://github.com/elpublica/PortfolioWeb.git)

# 2. Entrar a la carpeta
cd PortfolioWeb

# 3. Instalar las dependencias de Node.js
npm install

# 4. Iniciar el servidor de desarrollo
npm run dev
